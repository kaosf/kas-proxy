# Ka's Proxy

Very easy and simple proxy server.

## Usage

Installation.

```sh
npm install -g https://github.com/kaosf/kas-proxy/tarball/v0.0.2
```

Usage.

```
echo '{"listen": 10000, "dest": 9000}' > config.json
kas-proxy
```
